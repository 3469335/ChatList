# История изменений

Все значимые изменения в проекте ChatList будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/),
и этот проект придерживается [Semantic Versioning](https://semver.org/lang/ru/).

## [1.0.0] - 2026-01-12

### Добавлено
- Первый релиз ChatList
- Графический интерфейс на PyQt5
- Поддержка множества AI моделей через OpenRouter, OpenAI, DeepSeek, Groq
- Сохранение промтов и результатов в SQLite базе данных
- Экспорт результатов в Markdown и JSON
- AI-ассистент для улучшения промтов
- Настройки темы (светлая/темная) и размера шрифта
- Управление моделями через интерфейс
- Автоматическое создание `.env` файла из шаблона
- Сохранение базы данных в AppData пользователя
- Инсталлятор Inno Setup с полной поддержкой удаления

### Исправлено
- Улучшена обработка ошибок OpenRouter API
- Исправлена ошибка "No cookie auth credentials found" с информативными сообщениями
- Исправлена потеря API ключей при установке через инсталлятор

### Технические детали
- Python 3.11+
- PyQt5 5.15.10
- SQLite для хранения данных
- Поддержка Windows 10+

- `Добавлено` - для новых функций
- `Изменено` - для изменений в существующей функциональности
- `Устарело` - для функций, которые скоро будут удалены
- `Удалено` - для удаленных функций
- `Исправлено` - для исправления ошибок
- `Безопасность` - в случае уязвимостей
