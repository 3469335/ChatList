# GitHub Pages для ChatList

Эта директория содержит файлы для GitHub Pages.

## Структура

- `index.html` - главная страница лендинга

## Настройка

1. Замените `{username}` и `{repo}` в `index.html` на ваши данные
2. Обновите ссылки на релизы, если нужно
3. Добавьте скриншоты приложения (опционально)

## Деплой

GitHub Pages автоматически обновляется при изменении файлов в директории `docs/` благодаря GitHub Actions workflow.
